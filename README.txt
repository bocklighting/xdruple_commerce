
xTuple's Drupal Commerce Override Module (xdruple_commerce)
-----------------------------------------------------------
by Ben Thompson (bendiy), ben@xtuple.com

The xDruple Commerce module add hooks to override some features in Drupal
Commerce to integrate with xTuple ERP.

Installation
------------

 * xDruple Commerce has the following dependencies. Download and install then from:
   RESCUED TODO: https://drupal.org/sandbox/bendiy/1804762
   Commerce https://drupal.org/project/commerce
   xTuple's Drupal Fields REST Storage TODO: https://github.com/xtuple/portal
 * Copy the whole xdruple_commerce directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate the xDruple Commerce module

Learn more at http://www.xtuple.org.
