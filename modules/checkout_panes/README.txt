
xTuple's Drupal Commerce Checkout Panes (xdruple_commerce_checkout_panes)
-----------------------------------------------------------
by Ben Thompson (bendiy), ben@xtuple.com

Provides xTuple integrated checkout panes for Drupal Commerce.

Installation
------------

 * Copy the whole xdruple_commerce directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate the xTuple's Drupal Commerce
   Checkout Panes module.

Learn more at http://www.xtuple.org.
